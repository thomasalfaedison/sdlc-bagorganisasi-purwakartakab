
\subsection*{2.10 Halaman Informasi}
\addcontentsline{toc}{subsection}{2.10 Halaman Informasi}

\noindent Untuk mengatur informasi yang tampil pada Landing Page Bagorganisasi, user klik Menu Informasi dan akan ditampilkan halaman Informasi seperti berikut:

\begin{figure}[H]
    \centering
    \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/informasi.png}
    \caption{Tampilan Menu Informasi}
    \label{fig:bagan-alir}
\end{figure}

\noindent Pada menu Informasi, ada beberapa fitur yang user dapat gunakan seperti lihat detail data, ubah data, dan hapus data. Berikut tahapan setiap fitur pada Menu Informasi:

\begin{enumerate}
    \item \textbf{Detail Informasi}\\
    Untuk melihat detail Informasi, klik ikon mata pada kolom tabel dan akan diarahkan ke halaman detail. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/icon-mata-informasi.png}
        \caption{Tampilan Icon Mata Informasi}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/detail-informasi.png}
        \caption{Tampilan Detail Informasi}
        \label{fig:bagan-alir}
    \end{figure}

    Pada halaman detail Informasi, user dapat melihat informasi alamat, no telp/WA, email, map, tautan X/Twitter, tautan Facebook, tautan Instagram, dan tautan Youtube. User juga dapat mengubah data Informasi dengan klik tombol "Sunting" dan kembali ke daftar Informasi dengan klik tombol "Kembali". 

    \item \textbf{Mengubah Informasi}\\
    Untuk mengubah data Informasi, klik ikon pensil pada kolom tabel dan akan diarahkan ke halaman untuk melakukan ubah data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/icon-pensil-informasi.png}
        \caption{Tampilan Icon Pensil Informasi}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=6.75cm]{media/menu-informasi/edit-informasi.png}
        \caption{Tampilan Form Edit Informasi}
        \label{fig:bagan-alir}
    \end{figure}

    Data yang dapat user ubah adalah alamat, no telp/WA, email, link map, tautan X/Twitter, tautan Facebook, tautan Instagram, dan tautan Youtube. Jika sudah, user klik tombol "Simpan" 
    
    \item \textbf{Hapus Informasi}\\
    Untuk menghapus data Informasi, klik ikon sampah pada kolom tabel dan akan muncul pesan untuk melakukan hapus data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/hapus-informasi.png}
        \caption{Tampilan Konfirmasi Hapus Informasi}
        \label{fig:bagan-alir}
    \end{figure}

    Jika data tersebut dihapus, maka akan muncul tombol untuk menambahkan data Informasi seperti gambar dibawah:

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/hapus-berhasil.png}
        \caption{Tampilan Konfirmasi Setelah Hapus Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Tambah Teks Depan}\\
    Untuk melakukan tambah Teks Depan dapat dilakukan dengan klik tombol “Tambah Tentang Kami.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/hapus-berhasil.png}
        \caption{Tampilan Tombol Tambah Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    Setelah klik tombol “Tambah Informasi, maka akan dialihkan ke halaman form tambah Teks Depan. User akan diminta untuk mengisi deskripsi, tautan/link, dan gambar. Jika sudah klik tombol “Simpan” dan data Tentang Kami akan tersimpan. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/tambah-informasi.png}
        \caption{Tampilan Form Tambah Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    Tentang Kami yang sudah disimpan akan tampil pada Landing Page seperti gambar dibawah:

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-informasi/informasi-hasil.png}
        \caption{Tampilan Hasil Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

\end{enumerate}
