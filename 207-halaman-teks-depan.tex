
\subsection*{2.7 Halaman Teks Depan}
\addcontentsline{toc}{subsection}{2.7 Halaman Teks Depan}

\noindent Untuk mengatur Teks Judul pada Landing Page Bagorganisasi, user klik Menu Teks Depan dan akan ditampilkan halaman Teks Depan seperti berikut:

\begin{figure}[H]
    \centering
    \includegraphics[width=12.45cm, height=5.75cm]{media/menu-teks-depan/teks-depan.png}
    \caption{Tampilan Menu Teks Depan}
    \label{fig:bagan-alir}
\end{figure}

\noindent Pada menu Teks Depan, ada beberapa fitur yang user dapat gunakan seperti tambah Teks Depan, lihat detail data, ubah data, dan hapus data. Berikut tahapan setiap fitur pada Menu Teks Depan:

\begin{enumerate}
    \item \textbf{Tambah Teks Depan}\\
    Untuk melakukan tambah Teks Depan dapat dilakukan dengan klik tombol “Tambah Teks Depan”.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-teks-depan/tombol-tambah-teks-depan.png}
        \caption{Tampilan Tombol Tambah Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    Setelah klik tombol “Tambah Teks Depan”, maka akan dialihkan ke halaman form tambah Teks Depan. User akan diminta untuk mengisi teks, sub teks, teks tombol, tautan/link, dan memilih status aktif. 
    
    Status Aktif berfungsi untuk menentukan teks mana saja yang perlu tampil pada halaman Landing Page Bagorganisasi. 
    
    Jika sudah klik tombol “Simpan” dan data Teks Depan terbaru akan tersimpan. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=7.75cm]{media/menu-teks-depan/form-tambah-teks-depan.png}
        \caption{Tampilan Form Tambah Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    Teks Depan yang sudah disimpan akan tampil pada Landing Page seperti gambar dibawah:

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/teks-depan-simpan.png}
        \caption{Tampilan Hasil Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Detail Teks Depan}\\
    Untuk melihat detail Teks Depan, klik ikon mata pada kolom tabel dan akan diarahkan ke halaman detail. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-teks-depan/icon-mata-teks-depan.png}
        \caption{Tampilan Icon Mata Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-teks-depan/detail-teks-depan.png}
        \caption{Tampilan Detail Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    Pada halaman detail Teks Depan, user dapat melihat informasi Teks, sub teks, teks tombol dan tautan/link. User juga dapat mengubah data Teks Depan dengan klik tombol "Sunting" dan kembali ke daftar Teks Depan dengan klik tombol "Kembali". 

    \item \textbf{Mengubah Teks Depan}\\
    Untuk mengubah data Teks Depan, klik ikon pensil pada kolom tabel dan akan diarahkan ke halaman untuk melakukan ubah data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-teks-depan/icon-pensil-teks-depan.png}
        \caption{Tampilan Icon Pensil Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=7.75cm]{media/menu-teks-depan/edit-teks-depan.png}
        \caption{Tampilan Form Edit Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    Data yang dapat user ubah adalah teks, sub teks, teks tombol, tautan/link, dan memilih status aktif. Jika sudah, user klik tombol "Simpan" 
    
    \item \textbf{Hapus Teks Depan}\\
    Untuk menghapus data Teks Depan, klik ikon sampah pada kolom tabel dan akan muncul pesan untuk melakukan hapus data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-teks-depan/hapus-teks-depan.png}
        \caption{Tampilan Konfirmasi Hapus Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

\end{enumerate}
