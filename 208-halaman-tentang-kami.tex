
\subsection*{2.8 Halaman Tentang Kami}
\addcontentsline{toc}{subsection}{2.8 Halaman Tentang Kami}

\noindent Untuk mengatur informasi tentang kami pada Landing Page Bagorganisasi, user klik Menu Tentang Kami dan akan ditampilkan halaman Tentang Kami seperti berikut:

\begin{figure}[H]
    \centering
    \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/tentang-kami.png}
    \caption{Tampilan Menu Tentang Kami}
    \label{fig:bagan-alir}
\end{figure}

\noindent Pada menu Tentang Kami, ada beberapa fitur yang user dapat gunakan seperti unduh gambar, lihat detail data, ubah data, dan hapus data. Berikut tahapan setiap fitur pada Menu Tentang Kami:

\begin{enumerate}
    \item \textbf{Unduh Gambar}\\
    Untuk melakukan tambah Tentang Kami dapat dilakukan dengan klik icon "Unduh" maka otomatis akan mendowload gambar yang dipilih.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/unduh-gambar.png}
        \caption{Tampilan Unduh Gambar Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Detail Tentang Kami}\\
    Untuk melihat detail Tentang Kami, klik ikon mata pada kolom tabel dan akan diarahkan ke halaman detail. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/icon-mata-tentang-kami.png}
        \caption{Tampilan Icon Mata Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/detail-tentang-kami.png}
        \caption{Tampilan Detail Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    Pada halaman detail Tentang Kami, user dapat melihat informasi deskripsi, tautan, gambar, dan unduh gambar. User juga dapat mengubah data Tentang Kami dengan klik tombol "Sunting" dan kembali ke daftar Tentang Kami dengan klik tombol "Daftar Tentang Kami". 

    \item \textbf{Mengubah Tentang Kami}\\
    Untuk mengubah data Tentang Kami, klik ikon pensil pada kolom tabel dan akan diarahkan ke halaman untuk melakukan ubah data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/icon-pensil-tentang-kami.png}
        \caption{Tampilan Icon Pensil Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=6.75cm]{media/menu-tentang-kami/edit-tentang-kami.png}
        \caption{Tampilan Form Edit Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    Data yang dapat user ubah adalah deskripsi, tautan, dan gambar. Jika sudah, user klik tombol "Simpan" 
    
    \item \textbf{Hapus Tentang Kami}\\
    Untuk menghapus data Tentang Kami, klik ikon sampah pada kolom tabel dan akan muncul pesan untuk melakukan hapus data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/hapus-tentang-kami.png}
        \caption{Tampilan Konfirmasi Hapus Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    Jika data tersebut dihapus, maka akan muncul tombol untuk menambahkan data Tentang Kami seperti gambar dibawah:

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=4.75cm]{media/menu-tentang-kami/hapus-berhasil.png}
        \caption{Tampilan Konfirmasi Setelah Hapus Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Tambah Teks Depan}\\
    Untuk melakukan tambah Teks Depan dapat dilakukan dengan klik tombol “Tambah Tentang Kami.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/hapus-berhasil.png}
        \caption{Tampilan Tombol Tambah Teks Depan}
        \label{fig:bagan-alir}
    \end{figure}

    Setelah klik tombol “Tambah Teks Depan”, maka akan dialihkan ke halaman form tambah Teks Depan. User akan diminta untuk mengisi deskripsi, tautan/link, dan gambar. Jika sudah klik tombol “Simpan” dan data Tentang Kami akan tersimpan. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/tambah-tentang-kami.png}
        \caption{Tampilan Form Tambah Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

    Tentang Kami yang sudah disimpan akan tampil pada Landing Page seperti gambar dibawah:

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-tentang-kami/tentang-kami-hasil.png}
        \caption{Tampilan Hasil Tentang Kami}
        \label{fig:bagan-alir}
    \end{figure}

\end{enumerate}
