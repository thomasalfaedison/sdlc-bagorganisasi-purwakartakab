
\subsubsection*{2.11.1 Halaman Admin}
\addcontentsline{toc}{subsubsection}{2.11.1 Halaman Admin}

\noindent Untuk mengelola data User Admin, klik Menu User lalu pilih Sub Menu Admin dan akan ditampilkan halaman User Admin seperti berikut:

\begin{figure}[H]
    \centering
    \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/admin.png}
    \caption{Tampilan Sub Menu Admin}
    \label{fig:bagan-alir}
\end{figure}

\noindent Pada Sub Menu Admin, ada beberapa fitur yang user dapat gunakan seperti filter data, tambah data, ubah password, ubah data, dan hapus data. Berikut tahapan setiap fitur pada Menu User:

\begin{enumerate}
    \item \textbf{Filter Username}\\
    Untuk filter data berdasarkan Username, klik kolom isian pada Card Filter, 
    lalu isikan Username salah satu User yang ingin ditampilkan dan klik tombol “Filter” seperti gambar 
    dibawah. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/filter-User.png}
        \caption{Tampilan Filter User}
        \label{fig:bagan-alir}
    \end{figure}
    
    \item \textbf{Tambah User}\\
    Untuk melakukan tambah User dapat dilakukan dengan klik tombol “Tambah User”.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/tombol-tambah-User.png}
        \caption{Tampilan Tombol Tambah User}
        \label{fig:bagan-alir}
    \end{figure}

    Setelah klik tombol “Tambah User”, maka akan dialihkan ke halaman form tambah User. User akan diminta untuk mengisi nama, username, dan password. Jika sudah klik tombol “Simpan” dan data User terbaru akan tersimpan. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/form-tambah-User.png}
        \caption{Tampilan Form Tambah User}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Ubah Password}\\
    Jika ada user yang lupa Password dan ingin me-reset atau mengubah Password, dapat melalui permintaan ke Admin. Untuk mengubah password akun Admin tertentu, klik ikon kunci pada kolom Set Password dan akan diarahkan ke halaman Ubah Password. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/icon-mata-User.png}
        \caption{Tampilan Icon Kunci User}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/detail-User.png}
        \caption{Tampilan Ubah Password}
        \label{fig:bagan-alir}
    \end{figure}

    Pada halaman ubah password, User akan diminta untuk mengisi password baru dan konfirmasi password baru. Jika sudah klik tombol “Simpan” dan password user tersebut sudah berhasil diubah.  

    \item \textbf{Mengubah User}\\
    Untuk mengubah data User, klik ikon pensil pada kolom tabel dan akan diarahkan ke halaman untuk melakukan ubah data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/icon-pensil-User.png}
        \caption{Tampilan Icon Pensil User}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=4.75cm]{media/menu-admin/edit-User.png}
        \caption{Tampilan Form Edit User}
        \label{fig:bagan-alir}
    \end{figure}

    Data yang dapat user ubah adalah nama dan username. Sebelum klik tombol "Simpan", kolom isian password harus diisi dengan password akun yang diubah, jika password salah maka gagal untuk mengubah data user yang sebelumnya sudah diisi. Jika sudah menyesuaikan data dan mengisi password yang benar, user klik tombol "Simpan" dan data akan berubah sesuai kolom isian.
    
    \item \textbf{Hapus User}\\
    Untuk menghapus data User, klik ikon sampah pada kolom tabel dan akan muncul pesan untuk melakukan hapus data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-admin/hapus-User.png}
        \caption{Tampilan Konfirmasi Hapus User}
        \label{fig:bagan-alir}
    \end{figure}

\end{enumerate}
