
\subsection*{2.5 Halaman Pencapaian}
\addcontentsline{toc}{subsection}{2.5 Halaman Pencapaian}

\noindent Pada menu Pencapaian, ada beberapa fitur yang user dapat gunakan seperti tambah Pencapaian, unduh gambar, lihat detail data, ubah data, dan hapus data. Berikut tahapan setiap fitur pada Menu Pencapaian:

\begin{enumerate}
    \item \textbf{Tambah Pencapaian}\\
    Untuk melakukan tambah Pencapaian dapat dilakukan dengan klik tombol “Tambah Pencapaian”.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-pencapaian/tombol-tambah-Pencapaian.png}
        \caption{Tampilan Tombol Tambah Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    Setelah klik tombol “Tambah Pencapaian”, maka akan dialihkan ke halaman form tambah Pencapaian. User akan diminta untuk mengisi judul dan upload gambar. Jika sudah klik tombol “Simpan” dan data Pencapaian terbaru akan tersimpan. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-pencapaian/form-tambah-Pencapaian.png}
        \caption{Tampilan Form Tambah Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    Data Pencapaian yang sudah disimpan akan tampil pada Landing Page seperti gambar dibawah:

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/pencapaian.png}
        \caption{Tampilan Hasil Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Unduh Gambar}\\
    Untuk melakukan tambah Pencapaian dapat dilakukan dengan klik icon "Unduh" maka otomatis akan mendowload gambar yang dipilih.
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-pencapaian/unduh-gambar.png}
        \caption{Tampilan Unduh Gambar Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    \item \textbf{Detail Pencapaian}\\
    Untuk melihat detail Pencapaian, klik ikon mata pada kolom tabel dan akan diarahkan ke halaman detail. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-pencapaian/icon-mata-Pencapaian.png}
        \caption{Tampilan Icon Mata Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=7.75cm]{media/menu-pencapaian/detail-pencapaian.png}
        \caption{Tampilan Detail Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    Pada halaman detail Pencapaian, user dapat melihat informasi judul, gambar Pencapaian, dan unduh gambar. User juga dapat mengubah data Pencapaian dengan klik tombol "Sunting" dan kembali ke daftar Pencapaian dengan klik tombol "Daftar Pencapaian". 

    \item \textbf{Mengubah Pencapaian}\\
    Untuk mengubah data Pencapaian, klik ikon pensil pada kolom tabel dan akan diarahkan ke halaman untuk melakukan ubah data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-pencapaian/icon-pensil-Pencapaian.png}
        \caption{Tampilan Icon Pensil Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=7.75cm]{media/menu-pencapaian/edit-Pencapaian.png}
        \caption{Tampilan Form Edit Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

    Data yang dapat user ubah adalah Judul dan Gambar. Jika sudah, user klik tombol "Simpan" 
    
    \item \textbf{Hapus Pencapaian}\\
    Untuk menghapus data Pencapaian, klik ikon sampah pada kolom tabel dan akan muncul pesan untuk melakukan hapus data. 

    \begin{figure}[H]
        \centering
        \includegraphics[width=12.45cm, height=5.75cm]{media/menu-pencapaian/hapus-Pencapaian.png}
        \caption{Tampilan Konfirmasi Hapus Pencapaian}
        \label{fig:bagan-alir}
    \end{figure}

\end{enumerate}
